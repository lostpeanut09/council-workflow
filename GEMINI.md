# Council Workflow (Project)

Regola: non saltare fasi.

Fasi obbligatorie:
1) Plan (docs/PLAN.md se presente)
2) Implement
3) Stage changes (git add -A)
4) Council review: /council:review (genera docs/REVIEW_KILO.md)
5) Apply: /council:review-apply (solo HIGH/MEDIUM)
6) Re-run tests
7) Summary + risks

Se /council:review non può essere eseguito: stop e spiega perché.
